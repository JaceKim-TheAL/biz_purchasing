## 해외구매대행사업 개요

### 운영 방식 비교
| 방식 | 초기자본 | 마진율 | 장점 | 단점 | 추천대상 | 
|------|--------|-------|------|------|--------|
| 직매입         | 300만원 이상 | 30~50% | 품질·배송 직접 관리 | 재고 부담, 포장·배송 직접 처리 | 자본 여유 있는 사람 |
| 위탁판매(무재고) | 0~50만원    | 15~25% | 재고 부담 없음, 다양한 상품 테스트 | 배송 지연, 품질 관리 어려움 | 초보자 | 
| 드롭쉬핑(해외)   | 50만원      | 40~60% | 해외 상품 판매 가능 | 통관·배송 지연, CS 부담 | 경험자 | 


### 플랫폼 선택
- 네이버 스마트스토어: 국내 최대 사용자 기반, 진입 장벽 낮음.
- 쿠팡: 로켓배송 등 강력한 물류 시스템, 판매자 지원 프로그램 활발.
- 자사몰(카페24, 고도몰 등): 브랜드 구축 가능, 마케팅·유입은 직접 해결해야 함.

### 준비 단계 
- 사업자 등록 : 부업이라도 정식 사업자 등록 필요.
- 상품 소싱 : 도매 사이트(도매꾹, 사입삼촌 등) 또는 해외 소싱.
- 상품 등록 & 상세페이지 제작 : 사진·설명·후기 관리가 핵심.
- 마케팅 : 블로그, 인스타그램, 유튜브 등 SNS 활용.
- 고객 관리 : 리뷰·CS 대응이 장기적인 신뢰 확보에 중요.

### 현실적인 수익 목표
- 위탁판매 기준 : 월 100만~200만 원 가능 (상품·마케팅 역량에 따라 변동).
- 직매입 전환 시 : 마진율 상승으로 월 200만~500만 원 이상도 가능

## 기본준비

✅ 스마트스토어 상품 등록 체크리스트

### 1. 기본 준비
- [ ] 사업자등록증, 통신판매업 신고증 준비
- [ ] 판매자 계정 로그인 및 스토어 개설 완료
- [ ] 판매 통장 연결 (정산 계좌)

### 2. 상품 정보 입력
- [ ] 상품명: 핵심 키워드 포함 (예: “여성 봄 원피스 긴팔 플라워 패턴”)
- [ ] 카테고리 선택: 최대한 정확하게 지정해야 검색 노출에 유리
- [ ] 가격 설정: 경쟁사 가격 조사 후 합리적 책정

### 3. 이미지 & 상세페이지
- [ ] 대표 이미지: 고화질, 깔끔한 배경
- [ ] 상세 이미지: 다양한 각도, 사용 예시 포함
- [ ] 상세 설명: 소재, 사이즈, 사용법, 주의사항 등 구체적으로 작성

### 4. 배송 & 재고
- [ ] 배송비 정책: 무료배송 vs 조건부 무료 → 고객 유입에 큰 영향
- [ ] 재고 관리: 위탁판매라면 공급처와 실시간 확인 필수
- [ ] 배송 기간 안내: 정확히 기재해야 CS 부담 줄어듦

### 5. 마케팅 & 리뷰
- [ ] 초기 리뷰 확보 전략 (지인 구매, 체험단 활용)
- [ ] 리뷰 답변은 빠르고 성실하게 → 신뢰도 상승
- [ ] 네이버 쇼핑 광고는 소액부터 테스트

<br/>
🎯 핵심 포인트 

- 상품명과 카테고리 정확성이 검색 노출의 핵심
- 상세페이지는 “고객이 궁금해할 모든 정보”를 담아야 CS 부담 감소
- 배송 정책은 고객 유입과 만족도에 직결

## 실전예시

### 📝 잘 팔리는 상품명 예시
- 여성 봄 원피스 긴팔 플라워 패턴 데일리룩
  - 계절 + 성별 + 아이템 + 특징 + 활용도
- 남성 운동화 런닝화 쿠션 좋은 가벼운 신발
  - 대상 + 제품군 + 기능성 키워드 강조
- 아동 유아 식판세트 BPA FREE 안전한 이유식 식기
  - 안전성 키워드 + 대상(아동/유아) + 용도
- 차량용 무선 충전 거치대 고속 충전 스마트폰 거치대
  - 기능 + 용도 + 핵심 키워드 반복으로 검색 노출 강화
- 캠핑용 접이식 테이블 휴대용 야외 피크닉 가구
  - 상황(캠핑/야외) + 특징(접이식/휴대용) + 제품군

### 💡 상품명 작성 공식
- 대상(누구를 위한 제품인지)
- 제품군(핵심 아이템)
- 특징(소재, 기능, 계절, 디자인)
- 활용도(어디서, 언제 쓰는지) 

이 4가지를 조합하면 검색에 잘 걸리고, 고객이 직관적으로 이해할 수 있는 상품명이 됩니다.
